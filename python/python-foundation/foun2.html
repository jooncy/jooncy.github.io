# 变量

变量是存储在内存中的值。 为了方便操作，数据分别用一个简单的名字代表，方便在接下来的程序中引用。变量就是代表某个数据(值)的名称。

# 变量命名规则

  • 必须由数字，字母，下划线的任意组合，且不能数字开头
  • 不能是Python中的关键字
  • 变量具有可描述性
  • 不能是中文

# 关键字(补充)

# 注释

注释是对代码的解释和说明，目的是让人们能够更加轻松地了解代码，和其他程序一样注释也不会出现在可执行程序中。因此在程序中根据需要添加任意的注释，而不必担心可执行的代码会膨胀。

注意

无需每一行都写注释，用到注释的是你牛逼的技巧解释，易忘的代码内容解释，理解很久才明白的代码内容解释

# Python 中常见的注释

  • 单行注释：#
  • 多行注释：""" 被注释内容 """

# 例-1：单行注释

# 这是个注释，能够注释单行
print("自强不息，厚德载物")

# 习题-6：用户登录，至少3次机会

for i in range(1,4):
    print("\033[31;0m您有3次机会重试，这是第%s次，次数用完程序将终止\033[0m\n"%(i))
    login_in = input("请输入账号：")
    passwd_in = input("请输入密码：")
    if  login != login_in and passwd == passwd_in:
        print("账号和密码不符 \n")

    if  login == login_in and passwd != passwd_in:
        print("密码输入错误！\n")
        continue
    if login != login_in and passwd != passwd_in:
        print("没有该账号密码，请核实 \n")
        continue
    if login == login_in and passwd == passwd_in:
