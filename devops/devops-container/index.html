# 序 前言

存货

这些年互联网一直鼓吹Docker，云计算，虚拟化云云... 实际真正全栈服务用上docker的公司并不多，都是诸如红帽子，OpenStack，阿里云，奇虎360这些公司。

Docker 是我出道的时候学习的技能，一直以来没有可应用的场景，直到2019年公司打算做全栈容器化并使用 Kubernetes 编排管理

# 技术选型是一件值得注意的事情，关乎着技术发展的战略

注意

有一个故事，技术人员发现了一个可以解决当前问题的新技术，于是兴冲冲用上了，运行上线后出现了比之前还大的问题，同时需要不断维护...

通常情况 下，一项新技术的诞生常常会伴随着媒体的大肆宣传和炒作，这使得用户很难看清技术的本质。 所以在做决策的时候需要慎重，应该通过常规流程选型、测试等验证是否符合当前公司技术环境
